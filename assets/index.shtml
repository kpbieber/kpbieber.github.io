Asset Definitions

Assets - A list and description of all your current Assets. You can change an asset name, description, and modify its attributes from this page. Each asset contains a link to view its personal dashboard as defined by its definition type.


Asset Definitions - A list of all current asset definitions you have defined. You can create new definitions as well as manage and configure current one. Specifically, you can add and remove alarms and data items for each definition. Here you can build dashboards for each definition type, which can then be viewed through an asset of that definition type.
